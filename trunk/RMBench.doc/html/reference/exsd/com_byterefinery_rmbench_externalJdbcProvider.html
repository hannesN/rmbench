External Jdbc Provider

Identifier:
com.byterefinery.rmbench.externalJdbcProvider

Since:
1.0.6

Description:
This extension point allows the definition of adapter classes which interface between RMBench and other plugins which maintain database connection information. One notable example being, of course, Eclipse DTP

Configuration Markup:

<!ELEMENT extension (provider)+>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>


<!ELEMENT provider EMPTY>

<!ATTLIST provider

id        CDATA #REQUIRED

name      CDATA #REQUIRED

className CDATA #REQUIRED

>

  • id -
  • name -
  • className - the class that implements the adapter interface. Must provide a zero-argument constructor
