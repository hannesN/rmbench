Model Exporter

Identifier:
com.byterefinery.rmbench.modelExporter

Since:
1.0

Description:
This extension point allows the definition of model exporters, which are used to export the model, or parts thereof, to arbitrary external formats

Configuration Markup:

<!ELEMENT extension (exporter+)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>


<!ELEMENT exporter EMPTY>

<!ATTLIST exporter

id            CDATA #IMPLIED

name          CDATA #REQUIRED

factory       CDATA #REQUIRED

description   CDATA #IMPLIED

wizardFactory CDATA #IMPLIED

>

an exporter provides a way to export a model, or selected model elements, to a file


  • id - optional unique id for this exporter extension
  • name - required name, which is presented to the user
  • factory - name of a class that implements the com.byterefinery.rmbench.extension.IModelExporter.Factory interface
  • description - optional description, which is displayed to the user along with the name
  • wizardFactory - name of a class that implements com.byterefinery.rmbench.extension.IModelExporterWizardFactory, which is responsible for creating configuration wizard pages for an exporter
