Image Exporter

Identifier:
com.byterefinery.rmbench.imageExporter

Since:
1.0

Description:
this extension point allows the definition of exporters that are responsible for exporting diagrams, or selected diagram elements, to image files

Configuration Markup:

<!ELEMENT extension (exporter+)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>

this extension point allows the definition of exporters that are responsible for exporting diagrams, or selected diagram elements, to image files



<!ELEMENT exporter EMPTY>

<!ATTLIST exporter

id          CDATA #IMPLIED

name        CDATA #REQUIRED

className   CDATA #REQUIRED

description CDATA #IMPLIED

>

an exporter provides a way to export a diagram, or selected diagram elements, to a file


  • id - the unique ID
  • name - the displayable and translateable name
  • className - the fully qualified name of a class that implements the com.byterefinery.rmbench.extension.IImageExporter interface
  • description - a displayable, translateable description

Examples: 

   <extension
         point="com.byterefinery.rmbench.imageExporter">
      <exporter
            className="com.mycorp.rmbench.imageext.FancyExporter"
            description="FANCY image"
            id="com.mycorp.rmbench.imageext.fancy"
            description="some fancy image format"
            name="FANCY"/>
   </extension>

API Information:
the class named in the className attribute must implement the com.byterefinery.rmbench.extension.IImageExporter interface

Supplied Implementation:
The RMBench commercial editions come with implementations for various popular export formats
