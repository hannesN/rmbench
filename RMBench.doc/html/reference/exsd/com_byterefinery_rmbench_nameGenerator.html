Name Generator

Identifier:
com.byterefinery.rmbench.nameGenerator

Since:
1.0

Description:
name generators are responsible for generating names for database objects

Configuration Markup:

<!ELEMENT extension (generator+)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>


<!ELEMENT generator EMPTY>

<!ATTLIST generator

id        CDATA #REQUIRED

name      CDATA #REQUIRED

className CDATA #REQUIRED

>

a generator for database object names


  • id - a unique identifier for this element
  • name - a displayable, translatable name for this generator
  • className - the fully qualified name of the implementing class, which must implement the com.byterefinery.rmbench.extension.INameGenerator interface

Supplied Implementation:
RMBench supplies one default name generator which uses a simple algorithm to generate unique names
