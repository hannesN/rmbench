Extension points

RMBench uses the eclipse mechanisms to allow for extending the base functionality. For this purpose, it provides a number of extension points that can be used e.g. to add native database mappings, JDBC drivers, and code generators. RMBench itself uses these extension points to provide reference implementations. Where these implementations are not sufficient for a given purpose, it sould be easy to amend them.

In case that you develop some extension that might also serve other users of RMBench, we would be happy to consider integrating it into the main code base, thus ensuring future maintenance and compatibility. Contact us to negotiate an appropriate recompensation
