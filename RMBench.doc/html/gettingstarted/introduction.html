Introduction

RMBench Database Modeller is a tool for graphically designing and maintaining relational database models. It provides a collection of views and editors that support the user in performing the following tasks:

  • create a database model by manipulating graphical representations of the underlying relational elements
  • view the database model through an arbitrary number of diagrams that can show different, possibly overlapping aspects of the model
  • reverse engineer existing databases into a RMBench model
  • forward engineer the model by generating SQL DDL scripts, which can then be executed directly from the script editor
  • perform a comparison between the model and an existing database, also generating DDL statements that modify the database to conform to the current state of the model

RMBench has been designed with a keen focus on usability and completeness in functionality. Many tasks can be executed with a keystroke, by the click of a mouse button or via drag-and-drop. The most important relational elements are supported, such as Tables, Foreign Keys, other Constraints and Indexes.

It should however be emphasized at this point that RMBench is geared more towards conceptional modeling than towards fully detailed database-specific physical modeling. The latter usually requires capabilities that do not lend themselves well to grapical modelling and are therefore better performed by specialized tools.

RMBench is available in 2 editions:

  • the community edition is free of charge
  • the professional edition comes at a small charge but adds significant functionality, and the assurance that the RMBench codebase will be further maintained professionally
